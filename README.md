# Online Voting System

A web-based Online Voting System developed using JSP, JDBC, and MySQL.

## Features
- Secure login for Admin and Voter
- One vote per voter restriction
- Admin can manage voters and candidates
- Real-time vote counting and result display

## Technologies Used
- Java
- JSP
- JDBC
- MySQL
- HTML, CSS
- Apache Tomcat

## How to Run
1. Import project into NetBeans / Eclipse
2. Configure Tomcat Server
3. Import database into MySQL
4. Update DB credentials
5. Run the project
